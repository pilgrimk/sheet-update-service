# Sheet Update Service

A Python-based service for updating structured Google Sheets using a custom mapping engine.

## Features

- FastAPI service layer
- Google Sheets integration via service account
- Dynamic sheet structure parsing
- Flexible label mapping with normalization
- Week-based column resolution
